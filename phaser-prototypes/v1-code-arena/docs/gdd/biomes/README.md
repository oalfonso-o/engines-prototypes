# Biomas

Cada bioma tiene su propia carpeta.

## Estructura estándar

Cada biome debería tener, como mínimo:

- `README.md`: overview del biome.
- `room-graph.md`: mapa interno validado y nombres de salas.

Y cuando el biome ya esté más desarrollado:

- `hub-and-intro.md` si tiene hub o staging narrativo propio.
- `environment-and-platform-language.md`
- `enemies-and-bosses.md`
- `rewards-teleports-and-onboarding.md`

Si un archivo todavía no existe en un biome, significa que ese contenido sigue pendiente de cierre.
