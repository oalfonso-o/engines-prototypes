Swamp Platformer

Primer nivel platformer en Phaser para este repo: terreno de pantano con tiles del pack local, plataformas, gravedad, salto y monedas para recoger.

Swamp tileset + Shinobi
`A/D` o flechas para mover, `W`, `Up` o `Space` para saltar, `R` para reset. Flujo estándar: `npm install` y `npm run dev`
