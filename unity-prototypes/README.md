# Unity Prototypes

Run Unity commands from this folder or via `make -C unity-prototypes ...` from repo root.

`make bootstrap-v1` prepares the first code-driven Unity prototype.

`make run-v1` bootstraps the scene and opens the project in the Unity Editor.

`make build-v1` builds the macOS player into `unity-prototypes/v1-code-arena/Builds/Mac/CanuterUnityV1.app`.

`make play-v1` builds that player and launches it.
