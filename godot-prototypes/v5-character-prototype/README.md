# canuter-v5-character-prototype

Isolated Godot 4 sandbox for testing a simple humanoid-like character built from geometric parts attached to a small `Skeleton3D`.

## Controls

- `WASD`: move
- `Space`: jump
- `G`: trigger a local explosion impulse near the character

## Run

```bash
cd /Users/oalfonso/pipprojects/canuter
make run-v5-character
```

## Test

```bash
cd /Users/oalfonso/pipprojects/canuter
make test-v5-character
```
