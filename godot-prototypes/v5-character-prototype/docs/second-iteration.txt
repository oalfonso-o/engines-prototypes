Create a new isolated Godot 4 prototype scene for a simple 3D character test. Do not modify the current player, grenade, or gameplay scenes yet. This must be a fully self-contained sandbox that I can open and run independently.

This is the most important architectural rule:

The character must have TWO completely separate layers:

1. PHYSICAL / GAMEPLAY LAYER
- This is the real gameplay body.
- Use `CharacterBody3D` as the root node.
- Add exactly ONE invisible collision shape for real movement and collisions.
- This collision shape must be a vertical `CapsuleShape3D` inside a `CollisionShape3D`.
- This capsule is the ONLY shape that defines:
  - contact with the floor
  - contact with walls
  - jump reset / ability to jump again
  - general collision against the environment
- The capsule must remain vertical at all times.
- The capsule must represent the full body volume from approximately foot level to head level.
- The capsule is NOT the visible torso.
- The visible meshes must NOT be used for collision.
- The character must never rely on the torso mesh or foot meshes for floor detection.
- The character must use the capsule collider for `is_on_floor()`, wall collisions, gravity, and jump logic.

2. VISUAL / ANIMATION LAYER
- This is only the visible dummy body.
- It must be separate from the physics collision capsule.
- Add a `Skeleton3D` as a child used only for posing the visual body.
- Add a very small set of bones, enough to control:
  - root
  - chest / torso
  - head
  - left_hand
  - right_hand
  - left_foot
  - right_foot
- Attach simple visible geometry to the bones using `BoneAttachment3D`:
  - torso = capsule mesh
  - head = sphere mesh
  - left hand = sphere mesh
  - right hand = sphere mesh
  - left foot = sphere mesh
  - right foot = sphere mesh
- These visual parts are only for display and animation.
- These visual parts must never be used as collision or support against the floor.
- The feet are decorative/visual and may move up, down, or sideways during animation without affecting gameplay collision.
- The torso mesh is decorative/visual and must never be treated as the contact shape against the floor.
