# V10 Doom Wall Lab

Fast visual comparison lab for no-external-assets Doom-style wall generation in Godot.

## Controls

- `WASD`: move
- `Shift`: sprint
- `Mouse`: look
- `Esc`: release mouse
- `Left click`: recapture mouse

## Run

```bash
make run-v10-wall-lab
```

Or open the project directly in Godot:

```bash
/Applications/Godot_mono.app/Contents/MacOS/Godot --path godot-prototypes/v10-doom-wall-lab
```
