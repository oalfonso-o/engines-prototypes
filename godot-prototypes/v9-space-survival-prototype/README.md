# canuter-v9-space-survival-prototype

2D fixed-camera space survival shooter prototype.

## Controls

- `W`: push the ship up to full forward thrust
- `A`: rotate left
- `D`: rotate right
- `S`: brake down toward a slower cruise speed
- `Space`: restart after game over

## Run

From the repo root:

```bash
make run-v9-space
```

## Tests

From the repo root:

```bash
make test-v9-space
```
